- Cursor 훅(`afterAgentResponse`)이 실행될 때, 코드 변경이 있으면 `docs/worklog.md` 하단에 자동 항목을 추가한다.

- 운영자는 PR/릴리즈 전 `context.md`, `plan.md`, `checklist.md`도 함께 맞춘다.
