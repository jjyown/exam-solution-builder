# 01. 표준 작업 프롬프트

아래 템플릿을 새 채팅 시작 시 그대로 사용합니다.

```text
다음 작업을 진행하세요.

목표:
- 크롭 시험지 문항별 해설 작성/수정
- 내보내기 전 자동 검수(OpenAI preflight + 규칙 게이트) 수행
- 실패 문항은 즉시 2차 풀이로 교정

필수 규칙:
- [문제] / [빠른 정답] / [해설] 순서 유지
- 객관식 정답 번호와 해설 결론 일치
- 포기/회피 문장 금지
- 도형 파일은 보조 그림으로만 사용(문항 집계 제외)

검수 루프:
1) 문항별 무결성 체크
2) 오답/형식 위반 문항 즉시 재작성
3) 합본 동기화
4) OpenAI preflight 실행 (기본 soft, 마감 시 strict)
5) export gate 통과 후 DOCX 생성

자동 진행 원칙:
- 위 검수 루프는 중요 필수 절차로 간주하고 사용자 재확인 없이 연속 진행
- 단, 개념 재판정 필요/대량 동시 불합격/게이트 반복 실패/DOCX 재발 오류/원본 손상은 즉시 중단 후 보고

최종 보고 형식:
- 수정한 문항 번호와 사유
- 남은 리스크
- 최종 생성 파일 경로
```
