# 04. 내보내기 전 최종 게이트

## 1) 합본 구조 체크

- [ ] 각 문항에 `[문제]`, `[빠른 정답]`, `[해설]` 존재
- [ ] 문항 번호 누락/중복 없음
- [ ] 빠른정답 요약 파일과 합본 정답 일치

## 2) 의미 품질 체크

- [ ] 문항별 결론 근거가 최소 1개 이상 명시됨
- [ ] 정답/해설 모순 없음
- [ ] 불필요한 장황함, 자기모순 문장 제거

## 3) 자동 검수 체크

- [ ] `build:md --preflight-openai` 실행
- [ ] `export_preflight_openai.md` 생성 여부 확인
- [ ] 마감 전에는 `--preflight-openai-strict` 재실행
- [ ] `explanationExportGate` 통과 확인

## 4) 최종 내보내기

- [ ] `npm run write-final-docx -- --latest` 성공
- [ ] 생성된 DOCX 파일명/시간 최신 확인
- [ ] 최종 보고: 수정 문항, 남은 리스크, 파일 경로
