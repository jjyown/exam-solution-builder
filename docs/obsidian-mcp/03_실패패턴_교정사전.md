# 03. 실패 패턴 교정 사전

## 패턴 A: 도형 파일이 문항으로 잘못 집계됨

- 증상: 문항 수가 실제보다 많아짐(예: 7문항인데 9문항 생성)
- 원인: `*_fig*.png`가 본문 문항처럼 처리됨
- 교정: `manifest.items.file` 기준으로 본문 문항만 수집

## 패턴 B: 정답/해설 결론 불일치

- 증상: `[빠른 정답]`은 ④인데 해설 마지막은 ②
- 교정: 해설 최종 결론 줄을 기준으로 정답 헤더를 동기화

## 패턴 C: 내보내기 게이트 통과를 위해 해설 품질이 훼손됨

- 증상: 수식/논리 설명이 과도한 평문화로 약화됨
- 교정: 게이트를 맞추되 핵심 수학 논리(조건, 식 변형, 결론 근거)는 유지

## 패턴 D: OpenAI preflight 실패로 전체 중단

- 증상: 키/네트워크 오류에서 작업 중지
- 교정: 기본은 soft-fail로 대체 리포트 저장 후 계속, 릴리즈 직전 strict로 재검증
