# 02. 문항별 검수 체크리스트

문항마다 아래 항목을 순서대로 확인합니다.

- [ ] `[문제]`, `[빠른 정답]`, `[해설]` 3헤더가 모두 존재한다.
- [ ] `[빠른 정답]` 값과 `[해설] 마지막 결론`이 일치한다.
- [ ] 객관식이면 정답 번호 표현(①~⑤ 또는 N번)이 명확하다.
- [ ] 해설 중 계산/식 전개가 결론과 모순되지 않는다.
- [ ] 포기/회피 문장이 없다.
- [ ] 수식/표기가 export gate 규칙과 충돌하지 않는다.
- [ ] 그림이 필요한 문항이면 해설 직후 보조 그림 배치가 맞다.

## 불합격 처리 규칙

- 항목 하나라도 실패하면 즉시 "불합격"으로 판정한다.
- 해당 문항은 2차 풀이로 재작성하고 합본까지 동기화한다.
- 재검수 후에만 다음 문항으로 진행한다.
