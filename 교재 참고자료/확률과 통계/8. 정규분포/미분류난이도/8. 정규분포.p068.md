71) [정답] 70
[해설]
두 확률변수 $X, Y$ 가 정규분포를 따르므로 $Z$ 가
표준정규분포를 따르는 확률변수일 때,
$\mathrm{P}(X \leq 0)=\mathrm{P}(Z \leq-m)$
$\mathrm{P}(Y \leq 0)=\mathrm{P}\left(Z \leq \frac{-\left(m^{2}+2 m+16\right)}{\sigma}\right)$
$\mathrm{P}(X \leq 0)=\mathrm{P}(Y \leq 0)$ 에서
$-m=\frac{-\left(m^{2}+2 m+16\right)}{\sigma}$
$\sigma=\frac{m^{2}+2 m+16}{m}=m+2+\frac{16}{m}$
$m>0, \frac{16}{m}>0$ 이므로
절대부등식의 성질에 의하여
$\sigma \geq 2+2 \sqrt{m \times \frac{16}{m}}=10$
$m=\frac{16}{m}$ 일 때, $\sigma$ 의 값이 최소이므로
$m_{1}=4$
$m=4$ 일 때, 두 확률변수 $X, Y$ 는 각각 정규분포 $\mathrm{N}\left(4,1^{2}\right)$,
$\mathrm{N}\left(40,10^{2}\right)$ 을 따르므로
$\mathrm{P}(X \geq 1)=\mathrm{P}\left(Z \geq \frac{1-4}{1}\right)=\mathrm{P}(Z \geq-3)$
$\mathrm{P}(Y \leq k)=\mathrm{P}\left(Z \leq \frac{k-40}{10}\right)$
$\mathrm{P}(X \geq 1)=\mathrm{P}(Y \leq k)$ 에서
$\mathrm{P}(Z \geq-3)=\mathrm{P}\left(Z \leq \frac{k-40}{10}\right)$
$=\mathrm{P}\left(Z \geq-\frac{k-40}{10}\right)$
$-3=-\frac{k-40}{10}$
따라서 $k=70$

$\mathrm{P}\left(Z \geq \frac{k-50}{\sigma}\right)=\mathrm{P}\left(Z \leq \frac{k-65}{2 \sigma}\right)$ 에서

73) [정답] (3)
[해설]
확률변수 $Y$ 의 평균을 $m_{1}$, 표준편차 $\sigma_{1}$ 이라 하자.
확률변수 $X$ 가 정규분포 $\mathrm{N}\left(m, \sigma_{1}^{2}\right)$ 을 따르고,
확률변수 $Y$ 가 정규분포 $\mathrm{N}\left(m, \sigma_{1}^{2}\right)$ 을 따른다.
$Z$ 를 만족하는 부분을 $\mathrm{N}\left(m, \sigma_{1}\right)$ 에 대하여
조건 (가)에서 모든 실수 $x$ 에 대하여
조건 (나)의 부포 $\mathrm{N}(Y \leq m)$ 에 대하여
$\mathrm{P}(X \leq x-3)=\mathrm{P}(Y \leq x+3)$ 이므로
$\mathrm{P}\left(Z \leq \frac{x-3-m}{\sigma}\right)=\mathrm{P}\left(Z \leq \frac{x+3-m}{\sigma}\right)$ 에서
$\frac{x-3-m}{\sigma}=\frac{x+3-m_{1}}{\sigma_{1}}$
(ㄱ)은 모든 실수 $x_{1}$ 에 대하여 성립하므로
$m_{1}=m+6, \sigma_{1}=\sigma_{1}$
조건 (나)의 $\mathrm{P}(Y \geq m)=0.9772$ 에서
$\mathrm{P}(Y \geq m)=\mathrm{P}\left(Z \geq \frac{m-(m-6)}{\sigma}\right)$
$=\mathrm{P}\left(Z \geq-\frac{\sigma}{6}\right)=\mathrm{P}\left(Z \leq \frac{\sigma}{\sigma}\right)$
$=\mathrm{P}(Z \leq 0)+\mathrm{P}\left(0 \leq Z \leq \frac{\sigma}{6}\right)$
$=0.5+\mathrm{P}\left(0 \leq Z \leq \frac{\sigma}{6}\right)=0.9772$
$\mathrm{P}\left(0 \leq Z \leq \frac{\sigma}{6}\right)=0.9972-0.5=0.4772$
이 때 $\mathrm{P}(0 \leq Z \leq 2)=0.4772$ 이므로
$\frac{\sigma}{\sigma}=2, \sigma=3$
따라서 확률변수 $X$ 는 정규분포 $\mathrm{N}\left(m, 3^{2}\right)$ 을 따르고,
확률변수 $Y$ 는 정규분포 $\mathrm{N}\left(m+6,3^{2}\right)$ 을 따른다.

\end{array}
\end{array}$$
