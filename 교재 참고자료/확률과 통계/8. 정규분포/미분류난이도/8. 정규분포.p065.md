출근 시간을 확률변수 $X$ 라 하면 $X$ 는 평균이 66.4 분, 표준편차가 15 분인 정규분포를 따른다.
즉, 확률변수 $X$ 는 $\mathrm{N}\left(66.4,15^{2}\right)$ 을 따른다.
$$\begin{array}{l}
\mathrm{P}(\mathrm{X} \geq 73)=\mathrm{P}\left(\frac{\mathrm{X}-66.4}{15} \geq \frac{73-66.4}{15}\right) \\
=\mathrm{P}(Z \geq 0.44)=0.33
\end{array}$$
(i) 출근 시간이 73 분 미만인 직원이 지하철을 이용할 사건을 $A$ 라 하면 출근 시간이 73 분 미만일 확률이 $1-0.33=0.67$ 이고 직원의 $20 \%$ 가 이용하였으므로 사건 $A$ 가 일어날 확률 $\mathrm{P}(\mathrm{A})$ 는
$$P(A)=0.67 \times \frac{20}{100}=0.134$$
(ii) 출근 시간이 73 분 이상인 직원이 지하철을 이용할 사건을 $B$ 라 하면 출근 시간이 73 분 이상일 확률은 0.33 이고 직원의 $40 \%$ 가 이용하였으므로 사건 $B$ 가 일어날 확률 $\mathrm{P}(\mathrm{B})$ 는
$$P(B)=0.33 \times \frac{40}{100}=0.132$$
(i), (ii)에서 임의로 선택한 1 명이 지하철을 이용할 확률은
$$\mathrm{P}(\mathrm{~A} \cup \mathrm{~B})=\mathrm{P}(\mathrm{~A})+\mathrm{P}(\mathrm{~B})=0.266$$

확률변수 $X$ 는 평균이 57 인 정규분포를 따르므로 $X$ 의
확률밀도함수를 $f(x)$ 라 하면 함수 $y=f(x)$ 의 그래프는 직선 $x=57$ 에 대하여 대칭이다.
$$\begin{aligned}
\mathrm{P}(47 \leq X \leq 51) & =\mathrm{P}(57-10 \leq X \leq 57-6) \\
& =\mathrm{P}(57+6 \leq X \leq 57+10) \\
& =\mathrm{P}(63 \leq X \leq 67)=\mathrm{P}(63 \leq X \leq a)
\end{aligned}$$

이므로 $a=67$
$$\begin{aligned}
\mathrm{P}(53 \leq X \leq 61) & =\mathrm{P}(57-4 \leq X \leq 57)+\mathrm{P}(57 \leq X \leq 57+4) \\
& =2 \mathrm{P}(53 \leq X \leq 57)=0.52
\end{aligned}$$

이므로 $\mathrm{P}(53 \leq X \leq 57)=0.26$
한편, $\mathrm{P}(X \geq b)=0.76>0.5$ 이므로 $b<57$
$$\begin{aligned}
\mathrm{P}(X \geq b) & =\mathrm{P}(b \leq X \leq 57)+\mathrm{P}(X \geq 57) \\
& =\mathrm{P}(b \leq X \leq 57)+0.5=0.76
\end{aligned}$$
$\mathrm{P}(b \leq X \leq 57)=0.76-0.5=0.26=\mathrm{P}(53 \leq X \leq 57)$
이므로 $b=53$
따라서 $a+b=67+53=120$

확률변수 $X$ 가 정규분포 $\mathrm{N}\left(m, \sigma^{2}\right)$ 을 따르므로 확률변수
$Z=\frac{X-m}{\sigma}$ 은 표준정규분포 $\mathrm{N}(0,1)$ 을 따른다.
$$\begin{array}{l}
\mathrm{P}(\mathrm{~m} \leq \mathrm{X} \leq \mathrm{m}+12)-\mathrm{P}(\mathrm{X} \leq \mathrm{m}-12) \\
\begin{array}{l}
=\mathrm{P}\left(\frac{m-m}{\sigma} \leq Z \leq \frac{(m+12)-m}{\sigma}\right)-\mathrm{P}\left(Z \leq \frac{(m-12)-m}{\sigma}\right) \\
=\mathrm{P}\left(0 \leq \mathrm{Z} \leq \frac{12}{\sigma}\right)-\mathrm{P}\left(\mathrm{Z} \leq-\frac{12}{\sigma}\right) \\
=\mathrm{P}\left(0 \leq \mathrm{Z} \leq \frac{12}{\sigma}\right)-\mathrm{P}\left(\mathrm{Z} \geq \frac{12}{\sigma}\right)=0.3664
\end{array}
\end{array}$$

한편, $\mathrm{P}\left(0 \leq \mathrm{Z} \leq \frac{12}{\sigma}\right)+\mathrm{P}\left(\mathrm{Z} \geq \frac{12}{\sigma}\right)=0.5$

이므로 (ㄱ), (ㄴ)에서
$$\mathrm{P}\left(0 \leq \mathrm{Z} \leq \frac{12}{\sigma}\right)=0.4332$$

즉, $\frac{12}{\sigma}=1.5$ 이므로 $\sigma=8$
