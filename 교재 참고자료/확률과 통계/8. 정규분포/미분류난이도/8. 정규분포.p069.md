확률변수 $X$ 는 $\mathrm{N}\left(8,3^{2}\right)$ 을 따르고, 확률변수 $Y$ 는 $\mathrm{N}\left(\mathrm{m}, \sigma^{2}\right)$ 을 따른다.
이때
$$\begin{aligned}
\mathrm{P}(4 \leq X \leq 8)+\mathrm{P}(Y \geq 8) & =\mathrm{P}\left(-\frac{4}{3} \leq Z \leq 0\right)+\mathrm{P}\left(Z \geq \frac{8-m}{\sigma}\right) \\
& =\frac{1}{2}
\end{aligned}$$

이므로 $\frac{8-m}{\sigma}=\frac{4}{3}$ 가 성립한다.
$$\therefore m=8-\frac{4}{3} \sigma$$

따라서
$$\begin{array}{l}
\mathrm{P}\left(Y \leq 8+\frac{2 \sigma}{3}\right)=\mathrm{P}\left(Z \leq \frac{8+\frac{2 \sigma}{3}-m}{\sigma}\right) \\
=\mathrm{P}\left(Z \leq \frac{8+\frac{2 \sigma}{3}-\left(8-\frac{4}{3} \sigma\right)}{\sigma}\right) \\
=\mathrm{P}(Z \leq 2) \\
=\mathrm{P}(Z \leq 0)+\mathrm{P}(0 \leq Z \leq 2) \\
=0.5+0.4772 \\
=0.9772
\end{array}$$

[해설]
확률변수 $X$ 는 정규분포 $\mathrm{N}\left(m_{1}, \sigma_{1}{ }^{2}\right)$ 을 따르고 확률변수
$Y$ 는 정규분포 $\mathrm{N}\left(m_{2}, \sigma_{2}{ }^{2}\right)$ 을 따를 때, 모든 실수 $x$ 에 대하여
$$\mathrm{P}(X \leq x)=\mathrm{P}(X \geq 40-x)$$

가 성립하므로 $\quad m_{1}=\frac{x+40-x}{2}=20$
또한 모든 실수 $x$ 에 대하여
$$\mathrm{P}(Y \leq x)=\mathrm{P}(X \leq x+10)$$
$Z_{1}=\frac{X-20}{\sigma_{1}}, Z_{2}=\frac{Y-m_{2}}{\sigma_{2}}$ 라 하면 확률변수 $Z_{1}, Z_{2}$ 는 표준정규분포 $\mathrm{N}(0,1)$ 을 따르므로
$$\mathrm{P}\left(Z_{2} \leq \frac{x-m_{2}}{\sigma_{2}}\right)=\mathrm{P}\left(Z_{1} \leq \frac{x+10-20}{\sigma_{1}}\right)$$

이때, 모든 실수 $x$ 에 대하여
$$\frac{1}{\sigma_{2}} x-\frac{m_{2}}{\sigma_{2}}=\frac{1}{\sigma_{1}} x-\frac{10}{\sigma_{1}}$$

이 성립하므로 $\quad \sigma_{1}=\sigma_{2}, m_{2}=10$
따라서
$$\mathrm{P}(15 \leq X \leq 20)=\mathrm{P}(5 \leq Y \leq 10)=\mathrm{P}(10 \leq Y \leq 15)$$

이므로
$$\begin{array}{l}
\mathrm{P}(15 \leq X \leq 20)+\mathrm{P}(15 \leq Y \leq 20) \\
=\mathrm{P}(10 \leq Y \leq 15)+\mathrm{P}(15 \leq Y \leq 20) \\
=\mathrm{P}(10 \leq Y \leq 20) \\
=\mathrm{P}\left(\frac{10-10}{\sigma_{2}} \leq Z_{2} \leq \frac{20-10}{\sigma_{2}}\right)
\end{array}$$

즉 $\mathrm{P}\left(0 \leq Z_{2} \leq \frac{10}{\sigma_{2}}\right)=0.4772$ 이므로
$$\begin{array}{l}
\frac{10}{\sigma_{2}}=2, \quad \sigma_{2}=5 \\
\therefore m_{1}+\sigma_{2}=20+5=25
\end{array}$$
