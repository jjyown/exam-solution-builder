96. 네 개의 비어 있는 상자 $\mathrm{A}, \mathrm{B}, \mathrm{C}, \mathrm{D}$ 가 있다. 각각의 상자에 최대 5 개의 공을 넣을 수 있을 때, 네 상자 $\mathrm{A}, \mathrm{B}, \mathrm{C}$, D 에 $n(1 \leq n \leq 20)$ 개의 공을 남김없이 나누어 넣는 경우의 수를 $f(n)$ 이라 하자. 다음은 $f(15)+f(14)+f(13)$ 의 값을 구하는 과정이다. (단, 공은 구별하지 않고, 공을 하나도 넣지 않은 상자가 있을 수 있다.)

네 상자 $\mathrm{A}, \mathrm{B}, \mathrm{C}, \mathrm{D}$ 에 $n$ 개의 공을 남김없이 나누어 넣는 경우의 수는 공이 5 개씩 모두 20 개가 들어 있는 네 상자 $\mathrm{A}, \mathrm{B}, \mathrm{C}, \mathrm{D}$ 에서 총 $20-n$ 개의 공을 꺼내는 경우의 수와 같다.
(i) $n=15$ 인 경우

공이 5 개씩 모두 20 개가 들어 있는 네 상자 $\mathrm{A}, \mathrm{B}$, $\mathrm{C}, \mathrm{D}$ 에서 총 5 개의 공을 꺼내는 경우의 수와 같으므로
$$f(15)=\text { (가) }$$
(ii) $n=14$ 인 경우

공이 5 개씩 모두 20 개가 들어 있는 네 상자 $\mathrm{A}, \mathrm{B}$, $\mathrm{C}, \mathrm{D}$ 에서 총 6 개의 공을 꺼내는 경우의 수와 같으므로
$$f(14)={ }_{4} \mathrm{H}_{6}-\text { (나) }$$
(iii) $n=13$ 인 경우

공이 5 개씩 모두 20 개가 들어 있는 네 상자 $\mathrm{A}, \mathrm{B}$, $\mathrm{C}, \mathrm{D}$ 에서 총 7 개의 공을 꺼내는 경우의 수와 같으므로
$$f(13)=\text { (다) }$$
(i), (ii), (iii)에 의해
$$\begin{array}{l}
f(15)+f(14)+f(13) \\
=\text { (가) }+\left({ }_{4} \mathrm{H}_{6}-\text { (나) }\right)+\text { (다) }
\end{array}$$

이다.
