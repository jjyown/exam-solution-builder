진수의 조건에서 $x>0$
$$\begin{array}{l}
\log _{\frac{1}{5}} 125 x \cdot \log _{5} \frac{x}{5} \geq 0 \text { 에서 } \\
\left(\log _{\frac{1}{5}} 125+\log _{\frac{1}{5}} x\right)\left(\log _{5} x-\log _{5} 5\right) \geq 0 \\
\therefore\left(-3-\log _{5} x\right)\left(\log _{5} x-1\right) \geq 0
\end{array}$$
$\log _{5} x=t$ 로 놓으면 $(-3-t)(t-1) \geq 0$
$(t+3)(t-1) \leq 0 \quad \therefore-3 \leq t \leq 1$
즉 $-3 \leq \log _{5} x \leq 1$ 이므로 $\log _{5} 5^{-3} \leq \log _{5} x \leq \log _{5} 5$
밑이 1 보다 크므로 $\frac{1}{125} \leq x \leq 5$
(ㄱ), (ㄴ)의 공통 범위를 구하면 $\frac{1}{125} \leq x \leq 5$
따라서 $\alpha=\frac{1}{125}, \beta=5$ 이므로
$\alpha \beta=\frac{1}{25}$

$\log _{\frac{1}{4}} x=t$ 로 놓으면 주어진 부등식은
$$t^{2}+a t+b<0$$
$\frac{1}{16}<x<16$ 에서
$\log _{\frac{1}{4}} 16<\log _{\frac{1}{4}} x<\log _{\frac{1}{4}} \frac{1}{16}$, 즉 $-2<\log _{\frac{1}{4}} x<2$
$\therefore-2<t<2$
해가 $-2<t<2$ 이고 $t^{2}$ 의 계수가 1 인 이차부등식은
$(t+2)(t-2)<0 \quad \therefore t^{2}-4<0$
이 부등식이 (ㄱ)과 일치해야 하므로 $a=0, b=-4$
$\therefore a+b=-4$

[해설]
진수의 조건에서 $x>0$
$\log _{2} x=t$ 로 놓으면 $t^{2}+t-2 \geq 0$

즉 $\log _{2} x \leq-2$ 또는 $\log _{2} x \geq 1$ 이므로

밑이 1 보다 크므로 $x \leq \frac{1}{4}$ 또는 $x \geq 2$

157) [정답] $0<x<1$ 또는 $x>8$
[해설]
진수의 조건에서
$$x>0, x^{3}>0 \therefore x>0$$
$\left(\log _{2} x\right)^{2}-\log _{2} x^{3}>0$ 에서 $\left(\log _{2} x\right)^{2}-3 \log _{2} x>0$
$\log _{2} x=t$ 로 놓으면 주어진 부등식은
$t^{2}-3 t>0, t(t-3)>0$
$\therefore t<0$ 또는 $t>3$
즉 $\log _{2} x<0$ 또는 $\log _{2} x>3$ 이므로
$\log _{2} x<\log _{2} 1$ 또는 $\log _{2} x>\log _{2} 2^{3}$
밑이 1 보다 크므로 $x<1$ 또는 $x>8$
(ㄱ), (ㄴ)의 공통 범위를 구하면
$0<x<1$ 또는 $x>8$
