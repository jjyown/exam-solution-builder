135) [정답] (5)
[해설]
$x^{x^{2}-8}<x^{2 x}$ 에서 $x=1$ 일 때 $1^{-7}=1^{2}$ 이므로 부등식이 성립하지 않는다.
(i) $0<x<1$ 일 때,
$$\begin{array}{l}
x^{2}-8>2 x \text { 이므로 } x^{2}-2 x-8>0 \\
(x+2)(x-4)>0 \\
\therefore x<-2 \text { 또는 } x>4
\end{array}$$

그런데 $0<x<1$ 이므로 부등식을 만족시키는 $x$ 가 존재하지 않는다.
(ii) $x>1$ 일 때,
$$\begin{array}{l}
x^{2}-8<2 x \text { 이므로 } x^{2}-2 x-8<0 \\
(x+2)(x-4)<0 \\
\therefore-2<x<4
\end{array}$$

그런데 $x>1$ 이므로 $1<x<4$
(i), (ii)에서 주어진 부등식의 해는 $1<x<4$ 이므로
$\alpha=1, \beta=4$
$\therefore \alpha+\beta=5$

[해설]

$\therefore \frac{1}{2}<s<4$
