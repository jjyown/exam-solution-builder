# 교재 참고자료 (실사용)

이 폴더는 해설 생성 시 실제로 참조되는 **운영용 OCR 자산**입니다.

## 이 폴더를 언제 쓰나

- 해설 생성 API가 `textbookReferenceSelector`를 통해 기본적으로 이 경로의 md를 읽습니다.
- 즉, **실제 문제 해설 품질에 직접 영향을 주는 폴더**입니다.

## 권장 구조

- `<단원>/<유형>/<난이도>/*.md`
- 현재 난이도는 `미분류난이도`가 많아도 동작에는 문제 없습니다.

## 테스트 산출물과 구분

- 테스트/튜닝 산출: `scratch/ocr-mapping-runs/`
- 실사용 자산: `교재 참고자료/`

`scratch/` 아래 산출물은 검증·비교용이며, 운영 입력으로 혼용하지 않습니다.
