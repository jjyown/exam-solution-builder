함수 $y = -2^{-3x + k} + 4^n = -2^{-3(x - \frac{k}{3})} + 4^n$의
[그림: 그래프]
그래프는 함수 $y = 2^{3x}$, 즉 $y = 8^x$의 그래프
를 원점에 대하여 대칭이동한 후, $x$축의 방
향으로 $\frac{k}{3}$만큼, $y$축의 방향으로 $4^n$만큼 평
행이동한 것이다.
함수 $y = -2^{-3x + k} + 4^n$의 그래프가 사분면 중 두 개의 사분면만을 지나
려면 원점을 지나야 하므로
$0 = -2^{k} + 4^n$에서
$2^{k} = 2^{2n}$
$k = 2n$
즉, $f(n) = 2n$
한편, $2^{f(m)} + 2^{f(m+1)} = 320$에서
$2^{f(m)} + 2^{f(m+1)} = 2^{2m} + 2^{2(m+1)}$
$= 2^{2m} + 2^{2} \times 2^{2m}$
$= 5 \times 2^{2m} = 320$
$2^{2m} = 64 = 2^{6}$
따라서 $2m = 6$이므로
$m = 3$

03
$a > 1$에서 $| \log_{a} x | = \begin{cases} -\log_{a} x & (0 < x < 1) \\ \log_{a} x & (x \ge 1) \end{cases}$
이므로 점 $A$의 $x$좌표는 $-\log_{a} x = k$에서
$\log_{a} x = -k$
$x = a^{-k}$
즉, $A(a^{-k}, k)$
점 $B$의 $x$좌표는 $\log_{a} x = k$에서
$x = a^{k}$
즉, $B(a^{k}, k)$
점 $P$는 직선 $y = k$가 $y$축과 만나는 점이므로
$P(0, k)$
조건 (가)에서 점 $A$는 선분 $PB$의 중점이므로 $\overline{PA} = \overline{AB}$에서
$a^{k} - a^{-k} = a^{k} - a^{-k}$
$a^{k} = 2a^{-k}$
$(a^{k})^{2} = 2$
$a > 1$에서 $a^{k} > 0$이므로 $a^{k} = \sqrt{2}$
한편, 직선 $OA$의 기울기는
$\frac{k - 0}{a^{-k} - 0} = k a^{k} = \sqrt{2} k$
직선 $OB$의 기울기는
$\frac{k - 0}{a^{k} - 0} = \frac{k}{a^{k}} = \frac{k}{\sqrt{2}}$
조건 (나)에서 두 직선 $OA$, $OB$의 기울기의 합이 $\frac{3\sqrt{2}}{8}$이므로
$\sqrt{2} k + \frac{k}{\sqrt{2}} = \frac{3\sqrt{2}}{2} k = \frac{3\sqrt{2}}{8}$
$k = \frac{1}{4}$
$k = \frac{1}{4}$을 $a^{k} = \sqrt{2}$에 대입하면
$a^{\frac{1}{4}} = \sqrt{2}$
$a = (\sqrt{2})^{4} = 4$
따라서 $a + k = 4 + \frac{1}{4} = \frac{17}{4}$
