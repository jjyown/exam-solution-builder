$\alpha + \beta = 6, \alpha \beta = 2$

17 수열
$1^2, \frac{1^2+2^2}{2}, \frac{1^2+2^2+3^2}{3}, ..., \frac{1^2+2^2+3^2+...+10^2}{10}$
의 일반항은
$a_n = \frac{1^2+2^2+3^2+...+n^2}{n}$
$= \frac{\frac{n(n+1)(2n+1)}{6}}{n}$
$= \frac{(n+1)(2n+1)}{6}$

이고 마지막 항은 제10항이다.
따라서
$1 + \frac{1^2+2^2}{2} + \frac{1^2+2^2+3^2}{3} + ... + \frac{1^2+2^2+3^2+...+10^2}{10}$
$= \sum_{k=1}^{10} a_k$
$= \sum_{k=1}^{10} \frac{(k+1)(2k+1)}{6}$
$= \frac{1}{6} \sum_{k=1}^{10} (2k^2+3k+1)$
$= \frac{1}{6} (2 \sum_{k=1}^{10} k^2 + 3 \sum_{k=1}^{10} k + \sum_{k=1}^{10} 1)$
$= \frac{1}{6} (2 \times \frac{10 \times 11 \times 21}{6} + 3 \times \frac{10 \times 11}{2} + 10)$
$= \frac{315}{2}$

18 수열 $1, 1+3, 1+3+5, 1+3+5+7, ...$의 일반항은 첫째항이
1이고 공차가 2인 등차수열의 첫째항부터 제n항까지의 합이므로
$\frac{n\{1+(2n-1)\}}{2} = n^2$
그러므로 수열 $\{a_n\}$의 일반항은
$a_n = n^2(3n-2)$
$a_m = 2800$에서
$m^2(3m-2) = 2800$
$3m^3 - 2m^2 - 2800 = 0$
$(m-10)(3m^2 + 28m + 280) = 0$
이때 이차방정식 $3m^2 + 28m + 280 = 0$의 판별식을 $D$라 할 때,
$\frac{D}{4} = 14^2 - 3 \times 280 = -644 < 0$
이므로 $3m^2 + 28m + 280 = 0$은 실근을 갖지 않는다.
그러므로 $m = 10$
따라서
$\sum_{k=1}^{10} a_k = \sum_{k=1}^{10} k^2(3k-2)$
$= \sum_{k=1}^{10} (3k^3 - 2k^2)$
$= 3 \sum_{k=1}^{10} k^3 - 2 \sum_{k=1}^{10} k^2$
$= 3 \times (\frac{10 \times 11}{2})^2 - 2 \times \frac{10 \times 11 \times 21}{6}$
$= 8305$

19 $a_k = 2n-k$라 하면
