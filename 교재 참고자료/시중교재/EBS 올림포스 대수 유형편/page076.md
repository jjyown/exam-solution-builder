04 사인법칙과 코사인법칙
01 사인법칙
삼각형 ABC의 외접원의 반지름의 길이를 R이라 하면
$\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R$
`<보기>` 삼각형 ABC에서 ∠A, ∠B, ∠C의 크기를 각각 A, B, C로 나타내고,
이들의 대변의 길이를 각각 a, b, c로 나타내기로 한다.
</보기>
02 코사인법칙
삼각형 ABC에서
(1) $a^2 = b^2 + c^2 - 2bc \cos A$
(2) $b^2 = c^2 + a^2 - 2ca \cos B$
(3) $c^2 = a^2 + b^2 - 2ab \cos C$
Ⅱ. 삼각함수
[그림: 삼각형 ABC]
[그림: 삼각형 ABC]
• 삼각형 ABC에서 A=45°, B=30°,
a=2√2일 때, b의 값을 구하면
$\frac{a}{\sin A} = \frac{b}{\sin B}$ 에서
$\frac{2\sqrt{2}}{\sin 45^\circ} = \frac{b}{\sin 30^\circ}$
$\frac{2\sqrt{2}}{\frac{\sqrt{2}}{2}} = \frac{b}{\frac{1}{2}} \Rightarrow b=2$
• 삼각형 ABC에서 A=60°, b=2,
c=3일 때, a의 값을 구하면
$a^2 = b^2 + c^2 - 2bc \cos A$
$= 2^2 + 3^2 - 2 \times 2 \times 3 \times \cos 60^\circ$
$= 4 + 9 - 2 \times 2 \times 3 \times \frac{1}{2}$
$= 7$
a>0이므로 $a = \sqrt{7}$
03 사인법칙과 코사인법칙의 변형
(1) 사인법칙의 변형
① 삼각형 ABC의 외접원의 반지름의 길이를 R이라 하면
$\sin A = \frac{a}{2R}, \sin B = \frac{b}{2R}, \sin C = \frac{c}{2R}$
② a : b : c = sin A : sin B : sin C
(2) 코사인법칙의 변형
삼각형 ABC에서
① $\cos A = \frac{b^2 + c^2 - a^2}{2bc}$ ② $\cos B = \frac{c^2 + a^2 - b^2}{2ca}$ ③ $\cos C = \frac{a^2 + b^2 - c^2}{2ab}$
04 삼각형의 넓이
삼각형 ABC에서 두 변의 길이와 그 끼인각의 크기가 주어질 때, 삼각형 ABC
의 넓이를 S라 하면
$S = \frac{1}{2}bc \sin A = \frac{1}{2}ca \sin B = \frac{1}{2}ab \sin C$
`<보기>` 그림과 같은 사각형 ABCD에서 두 대각선의 길이가 각각 a, b이고, 두 대각
선이 이루는 각의 크기가 θ일 때, 사각형 ABCD의 넓이를 S라 하면
$S = \frac{1}{2}ab \sin \theta$
</보기>
[그림: 사각형 ABCD]
• 삼각형 ABC에서 A=30°, b=3,
c=4일 때, 넓이 S의 값을 구하면
$S = \frac{1}{2}bc \sin A$
$= \frac{1}{2} \times 3 \times 4 \times \sin 30^\circ$
$= \frac{1}{2} \times 3 \times 4 \times \frac{1}{2}$
$= 3$
