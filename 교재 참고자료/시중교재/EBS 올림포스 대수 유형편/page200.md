01 삼각형 ABC에서 사인법칙에 의하여
$\frac{BC}{sin A}=\frac{AB}{sin C}$
$\frac{2\sqrt{6}}{sin \frac{\pi}{3}}=\frac{4}{sin C}$, $\frac{2\sqrt{6}}{\frac{\sqrt{3}}{2}}=\frac{4}{sin C}$
$\frac{4\sqrt{2}}{2}=\frac{4}{sin C}$
$sin C=\frac{4}{4\sqrt{2}}=\frac{\sqrt{2}}{2}$

이때 $0 < \angle C < \frac{\pi}{3}$이므로 $\angle C = \frac{\pi}{4}$
따라서 $\angle B = \pi - (\angle A + \angle C) = \pi - (\frac{2}{3}\pi + \frac{\pi}{4})=\frac{\pi}{12}$

답 $\frac{\pi}{12}$

02 삼각형 ABC의 외접원의 반지름의 길이를 $R$이라 하면
사인법칙에 의하여
$\frac{BC}{sin A}=\frac{AB}{sin C}=2R$
$\frac{BC}{sin 45°}=\frac{3}{sin 30°}=2R$, $\frac{BC}{\frac{\sqrt{2}}{2}}=\frac{3}{\frac{1}{2}}=2R$
이므로 $BC=3\sqrt{2}$, $R=3$
이때 $\overline{BO}=R=3$이므로
$\overline{BC} \times \overline{BO}=3\sqrt{2} \times 3 = 9\sqrt{2}$

답 ②

03 $0 < A < \pi$이므로 $sin A = \sqrt{1-cos^2 A} = \sqrt{1-(\frac{2}{3})^2}=\frac{\sqrt{5}}{3}$

삼각형 ABC의 외접원의 반지름의 길이를 $R$이라 하자.

삼각형 ABC의 외접원의 넓이가 36$\pi$이므로
$\pi R^2 = 36\pi$
$R > 0$이므로 $R=6$
삼각형 ABC에서 사인법칙에 의하여
$\frac{BC}{sin A}=2R$, $\frac{BC}{\frac{\sqrt{5}}{3}}=2 \times 6$
따라서 $BC=12 \times \frac{\sqrt{5}}{3}=4\sqrt{5}$

답 $4\sqrt{5}$

04 원 C의 반지름의 길이를 $R$, C의 둘레의 길이를 $l$, 넓이를 $S$라 하면
$l=2\pi R$, $S=\pi R^2$이므로 $p=2R$, $q=R^2$
$p+q=2R+R^2=24$이므로
$R^2+2R-24=0$
$(R+6)(R-4)=0$
$R > 0$이므로 $R=4$
삼각형 ABC에서 사인법칙에 의하여
$\frac{BC}{sin A}=2R$
$\frac{BC}{sin 120°}=2 \times 4$, $\frac{BC}{\frac{\sqrt{3}}{2}}=8$
따라서 $BC=8 \times \frac{\sqrt{3}}{2}=4\sqrt{3}$

답 ⑤

05 원에 내접하는 사각형의 마주보는 두 각의 크기의 합은 $\pi$이므로
$\angle BCD = \pi - \angle DAB = \frac{\pi}{3}$
삼각형 BCD에서 사인법칙에 의하여
$\frac{BD}{sin (\angle BCD)}=\frac{BC}{sin (\angle CDB)}$
$\frac{BD}{sin \frac{\pi}{3}}=\frac{4\sqrt{3}}{sin \frac{\pi}{4}}$, $\frac{BD}{\frac{\sqrt{3}}{2}}=\frac{4\sqrt{3}}{\frac{\sqrt{2}}{2}}$
따라서 $\overline{BD} = 4\sqrt{6} \times \frac{\sqrt{3}}{2} = 6\sqrt{2}$

답 $6\sqrt{2}$

[다른 풀이]
원의 반지름의 길이를 $R$이라 하면 삼각형 ABD에서 사인법칙에 의하여
$\frac{BD}{sin (\angle DAB)}=2R$, $\frac{BD}{sin \frac{\pi}{3}}=2R$  …… ㉠
삼각형 BCD에서 사인법칙에 의하여
$\frac{BC}{sin (\angle CDB)}=2R$, $\frac{4\sqrt{3}}{sin \frac{\pi}{4}}=2R$  …… ㉡
㉠, ㉡을 연립하면
$\frac{BD}{sin \frac{\pi}{3}}=\frac{4\sqrt{3}}{sin \frac{\pi}{4}}$
따라서 $\overline{BD} = \frac{4\sqrt{3}}{\frac{\sqrt{2}}{2}} \times \frac{\sqrt{3}}{2} = 6\sqrt{2}$
