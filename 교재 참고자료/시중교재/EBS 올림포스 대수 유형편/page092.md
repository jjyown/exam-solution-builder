04 등차수열의 합
(1) 첫째항이 $a$, 제 $n$항이 $l$인 등차수열 $\{a_n\}$의 첫째항부터 제 $n$항까지의 합 $S_n$은
$$S_n = \frac{n(a+l)}{2}$$
(2) 첫째항이 $a$, 공차가 $d$인 등차수열 $\{a_n\}$의 첫째항부터 제 $n$항까지의 합 $S_n$은
$$S_n = \frac{n\{2a+(n-1)d\}}{2}$$
<참고> 첫째항이 $a$, 공차가 $d$인 등차수열 $\{a_n\}$의 첫째항부터 제 $n$항까지의 합 $S_n$은
$$S_n = \frac{n\{2a+(n-1)d\}}{2} = \frac{d}{2}n^2 + \frac{2a-d}{2}n$$
이때 $\frac{d}{2} = A, \frac{2a-d}{2} = B$라 하면 $S_n = An^2 + Bn$이다.
즉, $d \neq 0$일 때, $S_n$은 $n$에 대한 이차식이고 상수항이 $0$이다.

$\cdot l = a + (n-1)d$이므로
$a+l = 2a+(n-1)d$
05 수열의 합과 일반항 사이의 관계
수열 $\{a_n\}$의 첫째항부터 제 $n$항까지의 합을 $S_n$이라 하면
$$a_1 = S_1, a_n = S_n - S_{n-1} (n \geq 2)$$
가 성립한다.
<참고> 수열 $\{a_n\}$의 첫째항부터 제 $n$항까지의 합 $S_n$이 $S_n = pn^2+qn+r$ ($p, q, r$은 상수)일 때,
$a_1 = p+q+r$이고,
$$a_n = S_n - S_{n-1} = (pn^2+qn+r) - \{p(n-1)^2+q(n-1)+r\} = 2pn-p+q \ (n \geq 2)$$
이다. 즉,
$$a_1 = p+q+r, a_2 = 3p+q, a_3 = 5p+q, a_4 = 7p+q, \cdots$$
이므로 다음이 성립한다.
① $r = 0$이면 수열 $\{a_n\}$은 첫째항부터 공차가 $2p$인 등차수열을 이룬다.
② $r \neq 0$이면 수열 $\{a_n\}$은 둘째항부터 공차가 $2p$인 등차수열을 이룬다.
수열 $\{a_n\}$의 첫째항부터 제 $n$항까지의 합 $S_n$이
① $S_n = 3n^2 - n$일 때,
$$a_1 = S_1 = 3 \times 1^2 - 1 = 2$$이고
$$a_n = (3n^2 - n) - \{3(n-1)^2 - (n-1)\} = 6n - 4 \ (n \geq 2)$$
따라서 $a_n = 6n-4$
② $S_n = 3n^2 - n + 1$일 때,
$$a_1 = S_1 = 3 \times 1^2 - 1 + 1 = 3$$이고
$$a_n = (3n^2 - n + 1) - \{3(n-1)^2 - (n-1) + 1\} = 6n - 4 \ (n \geq 2)$$
따라서 $a_n = \begin{cases} 3 & (n=1) \\ 6n-4 & (n \geq 2) \end{cases}$

$\cdot$ 수열의 합 $S_n$과 일반항 $a_n$ 사이의 관계는 모든 수열에서 성립한다.
