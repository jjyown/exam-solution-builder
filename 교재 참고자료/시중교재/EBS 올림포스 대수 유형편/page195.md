71 $\frac{2}{3}\pi x=t$라 하면 $0 \le x \le 6$에서 $0 \le \frac{2}{3}\pi x \le \frac{3}{3}\pi \cdot 6$, 즉 $0 \le t \le 4\pi$이고
$|2 \sin \frac{2}{3}\pi x| \ge 1$
즉, $|\sin t| \ge \frac{1}{2}$
$0 \le t \le 4\pi$일 때, 이 부등식의 해는 함수 $y = |\sin t|$의 그래프가 직선
$y = \frac{1}{2}$과 만나거나 그 위쪽에 있는 $t$의 값의 범위이므로
$\frac{\pi}{6} \le t \le \frac{5}{6}\pi$ 또는 $\frac{7}{6}\pi \le t \le \frac{11}{6}\pi$ 또는 $\frac{13}{6}\pi \le t \le \frac{17}{6}\pi$ 또는
$\frac{19}{6}\pi \le t \le \frac{23}{6}\pi$
[그림: $y = |\sin t|$ 그래프]

$\frac{\pi}{6} \le \frac{2}{3}\pi x \le \frac{5}{6}\pi$ 또는 $\frac{7}{6}\pi \le \frac{2}{3}\pi x \le \frac{11}{6}\pi$ 또는 $\frac{13}{6}\pi \le \frac{2}{3}\pi x \le \frac{17}{6}\pi$ 또는 $\frac{19}{6}\pi \le \frac{2}{3}\pi x \le \frac{23}{6}\pi$
