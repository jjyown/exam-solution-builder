유형 14 삼각함수의 성질
$(\frac{n}{2}\pi \pm \theta)$의 삼각함수의 변형 (단, $n$은 정수이다.)

| $n$이 짝수 | $n$이 홀수 |
|---|---|
| $\sin (\frac{n}{2}\pi \pm \theta) \rightarrow \sin \theta$ | $\sin (\frac{n}{2}\pi \pm \theta) \rightarrow \cos \theta$ |
| $\cos (\frac{n}{2}\pi \pm \theta) \rightarrow \cos \theta$ | $\cos (\frac{n}{2}\pi \pm \theta) \rightarrow \sin \theta$ |
| $\tan (\frac{n}{2}\pi \pm \theta) \rightarrow \tan \theta$ | $\tan (\frac{n}{2}\pi \pm \theta) \rightarrow \frac{1}{\tan \theta}$ |

위와 같이 바꾼 후, 각 $\theta$를 예각으로 여길 때, 처음 $(\frac{n}{2}\pi \pm \theta)$의 삼각함수의 값의 부호가 양이면 +, 음이면 -를 붙인다.

유형 15 삼각함수의 성질의 활용
삼각함수의 성질과 삼각함수 사이의 관계, 삼각형의 성질 등
을 이용하여 문제를 해결한다.
(1) 삼각함수 사이의 관계
$\tan \theta = \frac{\sin \theta}{\cos \theta}$, $\sin^2 \theta + \cos^2 \theta = 1$
(2) 삼각형의 성질
삼각형 ABC에서
① 세 내각의 크기의 합: $A+B+C=\pi$
② $0 < A < \pi, 0 < B < \pi, 0 < C < \pi$이므로
$0 < \sin A < 1, 0 < \sin B < 1, 0 < \sin C < 1$
