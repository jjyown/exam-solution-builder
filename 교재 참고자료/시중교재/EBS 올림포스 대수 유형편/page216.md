02 등차수열 $\{a_n\}$의 첫째항을 $a$, 공차를 $d$라 하면

$a + 19d = 9$

$a_{100} - a_{99} + a_{98} - a_{97} + a_{96} - a_{95}$

$= d + d + d = 3d = 21$
에서 $d = 7$

$a_n = -124 + (n-1) \times 7 = 7n - 131$
따라서 $a_{25} = 7 \times 25 - 131 = 44$

03 등차수열 $\{a_n\}$의 공차가 4이므로 일반항을
$a_n = 4n + k$ ($k$는 상수)로 놓을 수 있다.
$3a_n - a_{n+1} = 3(4n + k) - \{4(n+1) + k\}$
$= 8n + 2k - 4$
이므로 수열 $\{3a_n - a_{n+1}\}$에서
제3항은 $8 \times 3 + 2k - 4$, 즉 $2k + 20$
제12항은 $8 \times 12 + 2k - 4$, 즉 $2k + 92$

$(2k + 20) + (2k + 92) = 0$에서

$a_n = 4n - 28$
