## OCR_본문

(i) $\log_b \frac{1}{2}$, 즉 $b = a^{\frac{1}{2}} = \sqrt{a}$일 때
$b=\sqrt{a}$를 $b=\sqrt{2a}$에 대입하면
$\sqrt{a} = \sqrt{2a}$
$\sqrt{a}(1-\sqrt{2a}) = 0$
$a = 0$ 또는 $a = \frac{1}{2}$

(ii) $\log_b b = 2$, 즉 $b = a^2$일 때
$b=a^2$을 $b=\sqrt{2a}$에 대입하면
$a^2 = \sqrt{2a}$
$a(a - \sqrt{2}) = 0$
$a = 0$ 또는 $a = \sqrt{2}$
이때 $a>1$이므로 (i), (ii)에서 $a = \sqrt{2}$이고 $b = a^2 = (\sqrt{2})^2 = 2$
따라서 $(ab)^2 = (\sqrt{2} \times 2)^2 = 8$

집합 $A = \{x | x는 |x| \le 4인 정수\}$에서 $n(A) = 9$
이때 집합 $X$가 집합 $A$의 공집합이 아닌 부분집합이므로
$0 < n(X) \le 9$
집합 $X$의 원소 중 양수인 것의 개수를 $p$, 음수인 것의 개수를 $q$라 하면 $p, q$는 모두 4 이하의 음이 아닌 정수이다.
양수 $x$의 제곱근 중 실수인 것은 $\pm \sqrt{x}$, 0의 제곱근 중 실수인 것은 0, 음수의 제곱근 중 실수인 것은 존재하지 않는다.
집합 $P = \{a | a는 x의 제곱근 중 실수, $x \in X$ \}에서
$0 \notin X$이면 $n(P) = 2p$, 즉 $n(P)$가 0 또는 짝수가 되어
$n(P) \times n(Q) = 9$를 만족시키지 않는다.
따라서 $0 \in X$이므로
$n(P) = 2p+1$
모든 실수 $x$에 대하여 $x$의 세제곱근 중 실수인 것의 개수는 1이므로
$Q = \{b | b는 x의 세제곱근 중 실수, $x \in X$ \}에서
$n(Q) = p+q+1$
$n(P) \times n(Q) = 9$에서
$(2p+1)(p+q+1) = 9$

방정식 ⑦을 만족시키는 4 이하의 음이 아닌 정수 $p, q$는 $p = 1, q = 1$뿐이다.
그러므로 $n(X) = 3$이고, 두 양수 $a, b$에 대하여 $X = \{-a, 0, b\}$이다.
집합 $A = \{x | -4 \le x \le 4, x는 정수\}$의 원소 중 음수의 개수는 4, 양수의 개수는 4이므로 가능한 집합 $X$의 개수는

06 두 점 $A(0, \log a), B(1, \log b)$를 지나는 직선의 방정식은
$y - \log a = \frac{\log b - \log a}{1-0} (x-0)$
$y = (\log \frac{b}{a}) \times x + \log a$

직선 ⑦이 점 (2, 2)를 지나므로
$2 = (\log \frac{b}{a}) \times 2 + \log a$
$= \log (\frac{b}{a})^2 + \log a$
$= \log (\frac{b^2}{a^2} \times a) = \log \frac{b^2}{a}$
즉, $\frac{b^2}{a} = 10^2$이므로
$a = \frac{b^2}{100}$

$a+b=24$에 $a = \frac{b^2}{100}$을 대입하면
$\frac{b^2}{100} + b = 24$
$b^2 + 100b - 2400 = 0$
$(b + 120)(b - 20) = 0$
$b > 0$이므로 $b = 20$
$a = 24 - b = 24 - 20 = 4$
따라서 $ab = 4 \times 20 = 80$
