57 박테리아의 처음 개체 수를 $a$, 매분 박테리아 개체 수의 일정한
증가율을 $r$ ($r > 0$)이라 하자.
10분이 지난 후의 개체 수는 처음 개체 수보다 $k$% 증가하였으므로
$a(1+r)^{10} = a + \frac{k}{100}a$
$a(1+r)^{10} = a(1 + \frac{k}{100})$
$(1+r)^{10} = \frac{100 + k}{100} \cdots$ ㉠
20분이 지난 후의 개체 수는 처음 개체 수보다 44% 증가하였으므로
$a(1+r)^{20} = a + \frac{44}{100}a$
$a(1+r)^{20} = a(1 + \frac{44}{100})$
$(1+r)^{20} = \frac{144}{100} \cdots$ ㉡
㉠의 양변을 제곱하면
$(1+r)^{20} = (\frac{100 + k}{100})^2$이므로 ㉡의 우변과 비교하면
$(\frac{100 + k}{100})^2 = \frac{144}{100} = (\frac{12}{10})^2$
$k > 0$이므로
$\frac{100 + k}{100} = \frac{12}{10}$
$100 + k = 120$
따라서 $k = 20$
