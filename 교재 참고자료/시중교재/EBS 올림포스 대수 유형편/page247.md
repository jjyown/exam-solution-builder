## OCR_본문

01
수열 $\{a_n\}$의 일반항이 $a_n=(-1)^{n^2}\sin\frac{n\pi}{6}$이므로 항을 나열하면
$a_1 = -\sin\frac{\pi}{6} = -\frac{1}{4}$
$a_2 = \sin\frac{2\pi}{6} = \sin\frac{\pi}{3} = \frac{3}{4}$
$a_3 = -\sin\frac{3\pi}{6} = -\sin\frac{\pi}{2} = -1$
$a_4 = \sin\frac{4\pi}{6} = \sin\frac{2\pi}{3} = \frac{3}{4}$
$a_5 = -\sin\frac{5\pi}{6} = -\frac{1}{4}$
$a_6 = \sin\frac{6\pi}{6} = \sin\pi = 0$
$a_7 = -\sin\frac{7\pi}{6} = -\left(-\frac{1}{4}\right) = \frac{1}{4}$
$\vdots$
이므로 모든 자연수 $n$에 대하여
$a_{6n-5} = -\frac{1}{4}, a_{6n-4} = \frac{3}{4}, a_{6n-3} = -1,$
$a_{6n-2} = \frac{3}{4}, a_{6n-1} = -\frac{1}{4}, a_{6n} = 0$
임을 알 수 있다.
수열 $\{a_n\}$의 첫째항부터 제$n$항까지의 합을 $S_n$이라 할 때,
$S_1 = a_1 = -\frac{1}{4}$
$S_2 = S_1 + a_2 = -\frac{1}{4} + \frac{3}{4} = \frac{1}{2}$
$S_3 = S_2 + a_3 = \frac{1}{2} + (-1) = -\frac{1}{2}$
$S_4 = S_3 + a_4 = -\frac{1}{2} + \frac{3}{4} = \frac{1}{4}$
$S_5 = S_4 + a_5 = \frac{1}{4} + \left(-\frac{1}{4}\right) = 0$
$S_6 = S_5 + a_6 = 0 + 0 = 0$
$S_7 = S_6 + a_7 = 0 + \left(-\frac{1}{4}\right) = -\frac{1}{4}$
$\vdots$
이므로 모든 자연수 $n$에 대하여
$S_{6n-5} = -\frac{1}{4}, S_{6n-4} = \frac{1}{2}, S_{6n-3} = -\frac{1}{2},$
$S_{6n-2} = \frac{1}{4}, S_{6n-1} = 0, S_{6n} = 0$
임을 알 수 있다.
따라서 $\sum_{k=1}^{m}a_k = S_m = -\frac{1}{2}$를 만족시키는 $m$은 $6n-3$의 꼴이어야 하므로 100 이하의 모든 자연수 $m$의 값의 합은
$$3+9+15+\cdots+99 = \frac{17\times(3+99)}{2} = 867$$

02
조건 (가)에서 $a_1, a_2, \cdots, a_{20}$ 중 항의 값이 $-1$인 항의 개수를 $a$, $0$인 항의 개수를 $b$, $1$인 항의 개수를 $c$, $2$인 항의 개수를 $d$라 하면 $a+b+c+d=20$ ($a\ge 1, b\ge 1, c\ge 1, d\ge 1$)

조건 (나)에서
$$\sum_{k=1}^{20}a_k^2 = 26$$
$a+c+4d=26$ ...... ㉠
$$\sum_{k=1}^{20}a_k^4 = 74$$
$a+c+16d=74$ ...... ㉡
㉠에서 ㉡을 변끼리 빼면
$12d=48, d=4$
이를 ㉠에 대입하면
$a+c=10, a=10-c$
이때 $a\ge 1, c\ge 1$에서 $1\le c\le 9$이고
$$\sum_{k=1}^{20}a_k = -a+c+2d=2c-2$$
이므로 이 값은 $a=1, c=9$일 때 최대이다.
따라서 구하는 최댓값은
$-1+9+8=16$

조건 (나)에서 모든 자연수 $n$에 대하여
$2a_{n+2}=a_n+a_{n+4}$
즉, $a_{n+2}-a_n=a_{n+4}-a_{n+2}$이므로 두 수열 $\{a_{2n-1}\}, \{a_{2n}\}$은 모두 등차수열을 이룬다.
이 두 수열의 공차를 각각 $d, d'$이라 하면
조건 (가)에서
$a_1+2a_2=a_3+2a_4$이므로
$a_1+2a_2=(a_1+d)+2(a_2+d')$
$d=-2d'$
$a_{12}=-3$에서
$a_2+5d'=-3$
$a_2=-3-5d'$ ...... ㉠
$a_{19}=1$에서
$a_1+9d=a_1-18d'=1$
$a_1=1+18d'$ ...... ㉡
$$\sum_{k=1}^{20}a_k=65$$
$(a_1+a_3+a_5+\cdots+a_{19})+(a_2+a_4+a_6+\cdots+a_{20})=65$
$$\frac{10(2a_1+9d)}{2} + \frac{10(2a_2+9d')}{2}=65$$
$(2a_1-18d')+(2a_2+9d')=13$
$2(a_1+a_2)-9d'=13$
위의 식에 ㉠, ㉡을 대입하면
$2\{(1+18d')+(-3-5d')\}-9d'=13$
$17d'=17, d'=1$
따라서 ㉡에서
$a_1=1+18\times 1=19$
