$f(x)=a \cos x-b$라 하자.
함수 $y=|f(x)|$의 그래프가 직선 $y=k$ ($k$는 자연수)와 만나는 서로
다른 점의 개수는 함수 $y=f(x)$의 그래프가 직선 $y=k$와 만나는 서로
다른 점의 개수와 함수 $y=f(x)$의 그래프가 직선 $y=-k$와 만나는
서로 다른 점의 개수의 합과 같다.
$-1 \le \cos x \le 1$이고 $a>0$이므로
$-a \le a \cos x \le a$이고 $-a-b \le f(x) \le a-b$
이때 $a-b \le 0$이면 $0 \le x \le 2\pi$일 때 $f(x) \le 0$이므로 함수 $y=f(x)$의
그래프는 직선 $y=3$, 직선 $y=5$와 만나지 않는다.
함수 $y=f(x)$의 그래프가 직선 $y=-3$과 만나는 서로 다른 점의 개수
는 최대 2개, 직선 $y=-5$와 만나는 서로 다른 점의 개수도 최대 2개
이므로 $m+n < 5$가 되어 조건을 만족시키지 않는다.
따라서 $a-b > 0$
또한 함수 $y=|f(x)|$의 그래프는 직선 $x=\pi$에 대하여 대칭이므로
$m+n=5$이려면 $m$, $n$ 중 하나만 홀수이어야 한다.
이때 함수 $y=|f(x)|$의 그래프와 직선 $y=5$가 만나는 서로 다른 점의
개수 $n$이 짝수이면 직선 $y=3$과 만나는 서로 다른 점의 개수 $m$도 짝
수가 되어 $m+n=5$를 만족시키지 않으므로 $n$은 홀수가 되어야 한다.
